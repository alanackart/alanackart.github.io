# Linux

## Ubuntu

Ubuntu安装后大约会占6G的空间， 60G磁盘就会占用到10%， 好伤心， 虽然可以在添加新磁盘， 但是只比较适合挂载到新用户目录上， 其他的操作方式会带来挺多问题， 虽然备份一下可以再移动到新磁盘， 但是软件太多的话， 还是不是很够用😂
